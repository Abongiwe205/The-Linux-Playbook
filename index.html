What is a virtual machine?

A virtual machine is a computer system created using software on one physical computer in order to emulate the functionality of another separate physical computer.

What is Operating systems?

An operating system is the most important software that runs on a computer. It manages the computer's memory and processes, as well as all of its software and hardware.

Different types of Operating Systems in a computer


  • Windows

  • MacOS

  • Linux

Different types of Linux systems

Ubuntu Linux
a Linux distribution based on Debian and composed primarily of free and open-source software.

Kali Linux
a Linux distribution based on Debian that is specifically designed for conducting Penetration Testing and security auditing tasks.

Mint Linux
an operating system for desktop and laptop computers. It is designed to work 'out of the box' and comes fully equipped with the apps most people need.
